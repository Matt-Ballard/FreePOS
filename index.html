Free POS

About This Site:

This web app aims to demonstrate the basic functionality of a 'till' (or 'cash register') - this is known as 'POS' (Point Of Sale).

How To Use The App:

You can add items to the POS in the section at the bottom of the page. Items added will appear in your 'Shop' screen as buttons.

Click on an item button to add that item to the basket. The basket represents the current transaction.

The display at the top shows the total bill for the current basket and the green button nearby finalizes the sale of the current basket.

The 'Quick Sale' payment method assumes that the exact payment has been tendered, so will not provide a prompt for change due.

The 'Basket View' button toggles a more detailed display, including info about the current basket & any change due.

The 'Payment View' button toggles futher payment options, including cash notes & a way to 'key-in' exact amounts.

The payment options also include a button to cancel the current sale, and a 'Sales Report' button which provides a summary of all trade from the session (an 'X report')

Refreshing the browser causes all the sales & product data from the current session to be lost, effectively resetting the POS to default.

Instead of manually inputting each item, there is an option to import items automatically from a text file.

To do this, simply provide a text file containing item data in the following 'comma-sperated' format: name,price,name,price... etc. An example of a properly formatted file might look like the line below:

Coffee,2.25,Cake,1.8,Mints,.90,Sandwich,4,Juice,1.05

The file should have the file extension.txt (use an app like notepad or save/rename a text file with the '.txt' suffix.
